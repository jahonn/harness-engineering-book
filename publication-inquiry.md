# 出版合作说明

本页用于说明《Harness Engineering》公开预览版的出版合作边界。公开预览版已经具备完整主稿结构、七编四十二章正文、尾注稿和 42 张工程图，中文字符超过 50 万；但它仍不是最终付印定稿。

本书适合以下出版方向：AI 工程、智能体系统、coding-agent 产品、企业内部 Agent OS、研发效能平台、AI 安全治理与软件工程实践。若出版社需要评估选题，可先阅读导读、前言、全书目录、第一章、第八章、第十一章、第四十章和注释与参考文献转换稿。

正式出版前仍需完成外部专家审稿、出版社体例确认、版权引用终审、正式开本打样、前后辅文确认和合同权利字段确认。公开预览版不包含尚未获得外部授权的内部材料；若后续进入出版流程，可在授权保留版和公开来源版之间做最终选择。

可洽谈事项包括简体中文版出版、电子书、有声书、课程或培训改编、营销摘录、公开样章、图表复用和海外或繁体中文版权利。具体授权范围以正式合同为准。

若通过本 GitBook 关联仓库或作者公开渠道联系，请在来信中说明出版社、编辑姓名、联系方式、感兴趣的出版形态、预计审读范围和是否需要 Word 全稿、样章包、图表清单或版权预审材料。收到明确出版沟通后，可提供 F1 送审材料、样章包、整稿 Word 审阅件和生产体例决策表。
