# Harness Engineering

《Harness Engineering》是一部面向 AI 智能体、coding-agent 产品和企业内部智能体平台建设的中文专业书稿。本公开预览版用于 GitBook 发布与选题沟通，采用已脱敏稿件，不包含尚未获得外部授权的内部材料。

本书讨论的 harness，是基础模型与真实任务环境之间的运行基底。它通过上下文、工具、权限、状态、观测、评测和反馈机制，把模型能力转化为可验证、可治理、可恢复的系统行为。

当前版本包含前言、全书目录、七编四十二章正文、注释与参考文献转换稿，以及 42 张工程图。公开预览版中文字符超过 50 万，满足长篇专业书稿的基本篇幅要求。

出版合作、审稿意见和版权事项以正式沟通材料为准；GitBook 版本只作为公开阅读与选题沟通入口。出版合作边界见[出版合作说明](publication-inquiry.md)。
